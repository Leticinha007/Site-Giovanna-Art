# Fonseccart — Site de Portfólio

**[fonseccart.vercel.app](https://fonseccart.vercel.app/)**

Site de portfólio desenvolvido para **Giovanna Fonseca**, artista e designer gráfica. Apresenta seus serviços, informações de contato e links para redes sociais.

## Tecnologias

- HTML5
- CSS3
- JavaScript (vanilla)
- Font Awesome 6
- Google Fonts (Montserrat)

## Estrutura

```
├── index.html       # Página principal
├── style.css        # Estilos
├── script.js        # Comportamento (menu hamburger, etc.)
├── images/          # Fotos dos serviços e perfil
└── logos/           # Logotipo da marca
```

## Seções

- **Início** — hero com chamada para ação
- **Sobre** — apresentação da artista e especialidades
- **Serviços** — grade de serviços oferecidos (papel, madeira, tela, ecobag, tecidos, paredes)
- **Contato** — telefone, e-mail, WhatsApp e redes sociais

## Deploy

Hospedado na [Vercel](https://vercel.com/). Qualquer push na branch `main` atualiza o site automaticamente.
